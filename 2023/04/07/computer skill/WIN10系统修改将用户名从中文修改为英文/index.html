【Computer Skill】Win10系统修改将用户名从中文修改为英文

2023-04-07 Computer Skill 590 字
Computer Skill Win10 用户名修改

win系统如果用户名一开始设置为中文名，那么C盘的该用户名文件夹会与后续安装的许多软件绑定在一起，特别是开源的软件，有时候会由于中文路径而使用不了，比如我最近在使用的Blender插件QuadRemesh便是如此。以下方法是我的处理方法，不过随意修改用户名有一定风险，还请谨慎处理，最好一开始将用户名设置英文，可以避免非常非常多的后续麻烦。

启动管理员账户

Win + X，选择“Windows PowerShell(管理员)打开终端

输入如下命令，启动管理员账户

net user administrator /active:yes

之后如需要关闭管理员账户，仅需重新输入如下命令即可

net user administrator /active:no

注销用户账户并使用管理员账户登录系统

注销当前的用户账户，之后使用管理员账户进入系统

将C盘用户文件夹进行重命名

将C:\用户\路径下，中文的用户名文件改为英文。

如果提示“文件夹正在使用”导致文件名无法修改，需打开任务管理器，选择性能-打开资源监视器

在“关联的句柄”处输入中文用户名，将搜索到的所有关联的模块右键选择结束进程，该过程可能会重启电脑，重启之后就可以修改用户文件名了

修改注册表

Win + R，输入regedit打开注册表编辑器，在搜索栏输入如下语句，点击回车

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList\

点击右边S-1-5开头的几个文件，查看右侧显示的ProfileImagePath，如果有包含中文用户名路径，双击将之改为英文用户名

创建软连接

Win + R，输入cmd打开命令行，输入如下指令（其中“中文用户名”和“英文用户名”自行替换），之后登录自己原本的账户即可，但还是多说一次，修改有一定风险，还是慎重行事，以上方式本人亲测可以，不对其他测试负责，本人操作过程也进行过多次保存以备万一。

mklink /d "C:\Users\中文用户名" "C:\Users\英文用户名"
