黑铁百科 基础篇001 坐标系与轴心点

  1. 坐标系

    • 全局世界坐标系
      相当于东南西北，即整个场景方位

    • 局部坐标系
      相当于前后左右，即所选物体的方位

      1）显示局部坐标系：

      2）使物体朝着局部坐标系运动：
      移动 G + X + X / G + Y + Y / G + Z + Z
      旋转 R + X + X / R + Y + Y / R + Z + Z
      …

    • 法向坐标系
      在编辑模式——面模式下，所选面的的z轴与该面的法线平行

    • 万象坐标轴
      即父子跟随运动，父级可以影响子级，子级无法影响父级。如xyz模式，z轴转动时会带动y轴，y轴转动时带动x轴，x轴转动时不会带动任何轴

    • 视图坐标系
      以当前观察者的视角为坐标系，z轴垂直于屏幕，左右为x轴，上下为y轴

    • 游标坐标系
      （在选择工具下）shift + 右键：移动游标
      使物体以游标坐标系为基准

    • 创建坐标系选定一个物体自身的坐标系作为基准，使其他物体以该物体的坐标系作为其自身的坐标系

    • 切换优先的坐标系

黑铁百科 基础篇002 吸附与衰减工具

  1. 吸附模式

    • 切换吸附模式

      1）快捷键：shift + tab
      2）临时切换：如移动物体时，按住ctrl，可进行临时状态切换（即若为开启吸附模式状态下，按住ctrl则为临时关闭吸附模式，若为关闭吸附模式状态下，按住ctrl则为临时开启吸附模式）
      3）吸附选项快捷键：shift + s

    • 被吸附对象属性
      1）增量——根据当前缩放级别对网格（栅格点）进行吸附
      绝对栅格对齐——使自带位移的物体，消除位移的影响，而吸附于网格（栅格点）
      2）顶点
      3）边
      4）面投射
      5）面最近
      6）体积
      7）边中点——顾名思义即某一条边的中点位置
      8）垂直交线

    • 吸附对象属性
      1）最近——距离被吸附对象最近的点
      2）中心——指当前的变换轴心点
      3）质心——基于原点计算的点
      4）活动项——当前选择对象（物体、点、线、面）的变换轴心点，优先级最高
      5）背面剔除——不视图背面部分产生吸附效果
      6）旋转对齐目标——吸附对象的z轴平行于被吸附对象的法向

      7）投影至自身——吸附对吸附对象自身有效
      8）项目的独立元素——多个吸附对象会根据其自身情况作用于被吸附对象

    • 吸附影响
      1）移动
      2）旋转
      3）缩放

    • 合并
      1）快捷键：M

  2. 衰减编辑——通过操作一个物体，连带影响其周围物体

    • 快捷键：O

    • 设置衰减范围：鼠标滚轮

    • 仅相连项：不相连接之间的对象不受影响

    • 从视角投影：仅根据当前视图情况下进行衰减编辑

黑铁百科 基础篇003 常规操作

  1. 名词

    • 活动项——多个选中对象之中最后选中的对象

    • 选中项——当前选中的对象，如果选中多个对象，则指多个对象根据其各自原点计算出来的质心

  2. 常用快捷键

    • 快速切换仅操作原点功能——ctrl + .

    • 吸附选项——shift + s
      1）选中项到游标（保持偏移）——选中的多个对象会根据其原点计算产生的质心，使之移动到游标的位置


      2）选中项到游标——选中的多个对象各自的原点所产生的质心，分别移动到游标的位置

    • 复制物体
      1）快捷键——shift + d

    • 合并物体
      选中多个物体，最后选中的物体的原点会作为这个合并的物体的新原点
      快捷键——ctrl + j

    • 分离物体
      进入编辑模式选择物体需要分离的部分
      快捷键——P

    • 选择相对元素——及与所选对象想连接的所有对象
      快捷键——L

    • 重复操作
      1）重复上一个步骤的操作
      快捷键——shift + r

    • 第一人称模式
      快捷键——shift + ~
      w——前进
      s——后退
      a——左
      d——右
      q——下降
      e——上升
      shift + 方位键——加速
      alt + 方位键——减速
      空格键——快速定位至视角中心
      退出第一人称模式——鼠标右键

    • 隐藏物体
      快捷键——H

    • 显示物体
      快捷键——alt + H

    • 设置父级目标
      选中多个物体，最后选中的物体将作为父级目标；父级会影响子级，子级不会影响父级
      快捷键——ctrl + p

    • 清除父子级关系
      快捷键——alt + p

  3. 常见案例

    • 使物体A围绕物体B旋转的方式
      1）选择物体A，开启仅操作原点功能

      2）先选择物体A，按住shift加选物体B，使物体B成为活动项，shift + s点击“选中项——>活动项”
      3）此时物体A的原点移动到了物体B的原点上，关闭仅操作原点功能（ctrl + .），于是物体A可围绕物体B旋转

黑铁百科 建模篇002 挤出工具

1. 挤出选区

  1. 面挤出

    • 快捷键:E（执行的操作为[挤出并移动]）默认按面法向进行移动，可点击x,y,z按轴向进行移动

    出现重叠面的情况：
    （1）使用快捷键E执行挤出并移动，在并未按左键进行确定的情况下，点击右键进行取消。此时仅取消移动操作，挤出操作完成执行，如此便产生了重叠面
    （2）此时进入透视模式（shift + Z）,使用点模式选择四个顶点，可在统计信息中看到有8个顶点
    （3）避免重叠面产生的正确做法是，在快捷键E执行时，先点击鼠标左键确定，再使用ctrl + Z进行撤销

    消除重叠面的方法：
    （1）使重叠点合并
    在透视模式下，使用点模式选择需要合并的点，快捷键M调出合并选项卡，选择[按距离]

    （2）融并正交边
    快捷键E挤出并移动，点击右键取消后，在右下角选项卡中勾选[融并正交边]
