黑铁百科001

启用Blender自带插件

  1. 节点工具


  1. 骨骼绑定工具


黑铁百科002

  1. 视角基本操作


1.1、旋转：

1）鼠标中键

2）漫游器

1.2、平移：

1）shfit + 鼠标中键

2）抓手

1.3、放大/缩小：

1）鼠标滚轮

2）放大镜

1.4、摄像机视角：

1）摄像机

2）数字键盘0

<1>在摄像机视角中，使摄像机随视图的变化而变化：按n键进入N面板，在视图选项卡中勾选“锁定相机到视图方位”


<2>将当前观察者视角设置为摄像机视角：ctrl + alt + 数字键盘0

1.5、透视/正交切换：

1）正交图

2）数字键盘5

1.6、切换四视图

1）ctrl + alt + Q


2）顶视图——数字键盘7

底视图——ctrl + 数字键盘7

3）前视图——数字键盘1

后视图——ctrl + 数字键盘1

4）右视图——数字键盘3

左视图——ctrl + 数字键盘3

5）视角翻转——数字键盘9（如顶视图切换为底视图、左视图切换为右视图）

1.7、聚焦选择物体

1）数字键盘.

<1>在四视图中使所有视图同步聚焦：ctrl + delete

1.8、聚焦并独显物体

1）数字键盘/

1.9、聚焦整个场景

1）数字键盘home

  1. 物体基本操作

2.1、移动

1）快捷键：G

2）移动归零：alt + G

3）在平面上移动：G + shift + z(在xy平面移动)

2.2、旋转

1）快捷键：R

2）旋转归零：alt + r

2.3、缩放

1）快捷键：S

2）缩放归零：alt + S

2.4、选择物体

1）全选：a

2）取消全选：aa / alt + a

黑铁百科004 吸附与衰减工具

切换吸附模式


快捷键：shift + tab

临时切换：如移动物体时，按住ctrl，可进行临时状态切换（即若为开启吸附模式状态下，按住ctrl则为关闭吸附模式）


国风水墨风格

通用快捷键：

视图&显示

鼠标中键：旋转
shift + 鼠标中键：平移
ctrl + Tab：模式切换
Tab：编辑模式
shift + z ：显示网格形态
alt + z：显示透视模式
ctrl + alt + 0：将当前视图设置为摄像机视图（需先选择摄像机物体）
ctrl + alt + Q：切换四视图
a：全选
aa：取消全选

n：显示工具栏小键盘home：显示整个场景小键盘.：聚焦视图到所选物体（需先选定物体）
ctrl + 小键盘.：四视图均聚焦到所选物体小键盘/：聚焦独显所选物体小键盘0：切换摄像机视角小键盘1：前视图
ctrl + 小键盘1：后视图小键盘3：右视图
ctrl + 小键盘3：右视图小键盘5：切换正交/透视视图小键盘7：顶视图
ctrl + 小键盘7：底视图小键盘9：视图翻转

操作

G：移动
G + X：X轴移动
G + Y：Y轴移动
G + Z：Z轴移动
Alt + G：物体位置回归零点
G + Shift + Z：在xy平面移动

ctrl + .：设置中心点

ctrl + l：关联传递数据选项卡
alt + n：法向选项卡

shift + tab：开启/关闭吸附

编辑模式快捷键：

ctrl + r：环切（loop cut）
shift + d：复制

雕刻模式

clay strips（粘条工具）：
ctrl + 鼠标左键：向下雕刻
shift + 鼠标左键：平滑
