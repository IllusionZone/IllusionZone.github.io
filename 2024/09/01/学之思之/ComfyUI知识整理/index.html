2024-09-01 学之思之 604 字
乐理

ComfyUI知识整理

安装及配置

主体安装

网址：https://github.com/search?q=comfyui&type=repositories

github官网搜索ComfyUI，进入页面后点击“Direct link to Download”进行下载，建议使用7.zip进行解压，否则可能会被吞掉某些文件

解压后可以选择CPU打开或GPU打开

插件安装

  1. 管理插件——便捷地管理、更新各种插件和模型

搜索ComfyUI-Manager，复制地址

在该路径下打开终端，输入git clone + 已复制的地址

安装后重启ComfyUI，且需等待其安装部分依赖后才能打开软件页面

安装成功后，右下角Manager即为插件界面，可以借助此插件安装及管理其他插件和各种模型

  1. 翻译插件——将软件界面翻译为中文等语言

点击Custom Nodes Manager，搜索AIGODLIKE-COMFYUI-TRANSLATION并安装，随后重启软件

在软件设置图标处，可切换语言。

  1. ControlNet插件

在绘市启动器 -> 版本管理 -> 安装新扩展，搜索comfyui_controlnet_aux进行安装

在\Comfyui\ComfyUI_windows_portable\ComfyUI\models\controlnet，打开终端，执行以下指令（此举为下载模型，亦可到对应的网址下载）

指令为：

git clone https://huggingface.co/lllyasviel/ControlNet-v1-1

网址为：
https://huggingface.co/lllyasviel/ControlNet-v1-1/tree/main

在\Comfyui\ComfyUI_windows_portable\ComfyUI\custom_nodes\comfyui_controlnet_aux\ckpts\Annotators，打开终端，执行以下指令（此举为下载预处理器，亦可到对应的网址下载）

指令为：

git clone https://huggingface.co/lllyasviel/Annotators

网址为：
https://huggingface.co/lllyasviel/Annotators/tree/main?clone=true

或可在网友收集好的网盘中下载后，放到\Comfyui\ComfyUI_windows_portable\ComfyUI\custom_nodes\comfyui_controlnet_aux\ckpts路径中

网盘为：
https://pan.quark.cn/s/8f873e906f36#/list/share
https://docs.stablediffusion.cn/?m=Article&a=index&id=4&aid=593415361399881728

  1. Florence2Run插件

在绘市启动器 -> 版本管理 -> 安装新扩展，搜索ComfyUI-Florence2进行安装

在\Comfyui\ComfyUI_windows_portable\ComfyUI\models新建文件夹并命名为LLM，进入LLM文件夹，打开终端，依次执行以下指令（此举为下载5个模型，亦可到对应的网址下载）

指令分别为：

git clone https://huggingface.co/microsoft/Florence-2-base
git clone https://huggingface.co/microsoft/Florence-2-base-ft
git clone https://huggingface.co/microsoft/Florence-2-large
git clone https://huggingface.co/microsoft/Florence-2-large-ft
git clone https://huggingface.co/HuggingFaceM4/Florence-2-DocVQA

网址分别为：
https://huggingface.co/microsoft/Florence-2-base
https://huggingface.co/microsoft/Florence-2-base-ft
https://huggingface.co/microsoft/Florence-2-large
https://huggingface.co/microsoft/Florence-2-large-ft
https://huggingface.co/HuggingFaceM4/Florence-2-DocVQA
