【学之思之】Hexo+Kaze+Github搭建个人博客

2022-12-21 学之思之 1.9k 字
个人博客 hexo 学之思之

本博客是采用Hexo架构，配合Kaze主题搭建，经GitHub和Gitee发布。

我也是偶然接触到，但这种既不需花钱，又没有审核，资源可以长久存在而不怕遗失，布局可以自定义而无需拘泥于一种风格的诱惑，对我而言实在太大了。于是义无反顾地一心扑了上来。

断断续续经过了一个月，期间生了点流行性小病，加上前后端知识的生疏，且网上资料不全，折腾许久才算勉强整出来这么点成果来。

当然，还存在很多瑕疵，只能是在后续的搭建过程中一边学习一边完善了。

此文也就此做下整理，记录本博客的搭建过程，以备日后之需。

账号申请及软件安装

更换高级Markdown渲染器

Hexo默认搭配的Markdown渲染器是hexo-renderer-marked,可选择更换为hexo-renderer-markdown-it，以实现更多功能以及更快的渲染速度

1、卸载hexo-renderer-marked

npm un hexo-renderer-marked --saved

2、安装hexo-renderer-markdown-it

npm i hexo-renderer-markdown-it --save

3、配置文件

在hexo根目录/_config.yml加入如下文本

markdown:
  preset: "default"
  render:
    html: true
    xhtmlOut: false
    langPrefix: "language-"
    breaks: true
    linkify: true
    typographer: true
    quotes: "“”‘’"
  enable_rules:
  disable_rules:
  plugins:
    - markdown-it-abbr
    - markdown-it-cjk-breaks
    - markdown-it-deflist
    - markdown-it-emoji
    - markdown-it-footnote
    - markdown-it-ins
    - markdown-it-mark
    - markdown-it-sub
    - markdown-it-sup
    - markdown-it-checkbox
    - markdown-it-imsize
    - markdown-it-expandable
    - name: markdown-it-container
      options: success
    - name: markdown-it-container
      options: tips
    - name: markdown-it-container
      options: warning
    - name: markdown-it-container
      options: danger
  anchors:
    level: 2
    collisionSuffix: ""
    permalink: false
    permalinkClass: "header-anchor"
    permalinkSide: "left"
    permalinkSymbol: "¶"
    case: 0
    separator: "-"

4、安装必要插件

npm i markdown-it-checkbox
npm i markdown-it-imsize
npm i markdown-it-expandable

5、重启电脑

主页配置

在切换白天/夜晚模式下更改作者头像

实现效果

"左边栏"的作者头像以及"关于"页面的作者图像会根据白天/黑夜模式，切换为对应的白天作者头像/黑夜作者头像

处理思路及相应操作如下：

1、在关于页面的作者头像标签加入id属性，方便后续进行头像更换控制，同时对该页面进行检测，根据其模式确定需要显示的作者头像

1） 文件路径：./themes/hexo-theme-Kaze/layout/about.ejs

2） 为标签添加id

<% if(theme.author_img) { %>
    <img 
      src="<%= url_for(theme.author_img) %>" 
      class="about-card-avatar"
      <!-- 此处设置标签卡id为isdark -->
      id="isdark" 
      alt="Blog Author Avatar" />
<% } %>

3） 在最后面添加代码，对页面进行检测，主要是通过全局变量isdark进行判断（该全局变量在下一个步骤创建），变量值为dark则显示黑夜模式的作者头像，变量为null或light则显示为白天模式的作者头像

<script>
// 变量imgSrc用于存储id为isdark的标签对象
  var imgSrc;
  if(window.isdark == 'dark'){
    // 如果为黑夜模式，则将img标签的src设置为黑夜图片路径
      imgSrc = document.getElementById('isdark')
      imgSrc.src = '/image/logo/黑夜作者头像.png'
  } else {
      imgSrc = document.getElementById('isdark')
      imgSrc.src = '/image/logo/白天作者头像.png'
  }
</script>

2、在head加载页设置一个全局变量，获取当前模式状态。

1） 文件路径：./themes/hexo-theme-Kaze/layout/_partial/head.ejs

2） 设置全局变量isdark，并在点击模式切换时，更改相应的作者头像

<% if(config.prismjs && config.prismjs.enable && theme.prism && theme.prism.darkTheme) { %>

    </script>

    <script>
      var setDarkmode = function(mode) {
      var setting = mode || localStorage.getItem('user-color-scheme');

      // setting获取到的值有三种情况，null,dark,light分别对应刚未点击模式切换按钮时的状态，模式切换按钮为黑夜时的状态以及模式切换按钮为白天时的状态，将此值赋予全局变量isdark
      window.isdark = setting;

      if(setting === getCssMediaQuery()) {
        //setting为light

        //setting为dark

      } else {
        //setting为null
        document.documentElement.removeAttribute('data-user-color-scheme');

        document.getElementById('theme-color').dataset.mode = getCssMediaQuery();
        
      }

      // setting 的值是Light、null和dark
      // 自定义添加的代码，当切换至其他页面时失效，所以在about和author页面添加同样的代码
      // 根据按键更换dark模式和light/null模式下的作者头像
      if(setting === "dark"){
        //判断是否进入“关于”页面，进入则同样修改“关于”页面的作者头像
        if(document.getElementById('isdark')){
          document.getElementById('isdark').setAttribute('src','/image/logo/logo3.png')
        }
        //修改左边栏作者头像
        document.getElementById('authorimg').setAttribute('src','/image/logo/logo3.png')
      } else {
        if(document.getElementById('isdark')){
          document.getElementById('isdark').setAttribute('src','/image/logo/logo2.png')
        }
        document.getElementById('authorimg').setAttribute('src','/image/logo/logo2.png')
      }

3、在关于页面进行检测，当处于黑夜/白天模式时，切换对于的作者头像

1） 文件路径：./themes/hexo-theme-Kaze/layout/about.ejs

2） 在文件最后添加如下代码

<script>
  var imgSrc;
  if(window.isdark == 'dark'){
      imgSrc = document.getElementById('isdark')
      imgSrc.src = '/image/logo/logo3.png'
  } else {
      imgSrc = document.getElementById('isdark')
      imgSrc.src = '/image/logo/logo2.png'
  }
</script>

内容页配置

基础配置

title: 标题
date: 发布日期
update: 更新日期
comments: 是否支持评论（true/false）
tags: 标签（[“标签名1”，“标签名2”，...]）
categories: 分类名
banner_img: 封面图片
excerpt：简介
